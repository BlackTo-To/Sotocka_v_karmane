# Веб‑приложение Соточка в кармане: ЕГЭ по математике

Интерактивное веб‑приложение, помогающее школьникам готовиться к ЕГЭ по математике (базовый и профильный уровни). Включает теорию, задачи с автоматической проверкой и статистику прогресса.

## Функциональные возможности

* **Каталог тем** — структурированные разделы по темам ЕГЭ:
    * Алгебра ($\text{уравнения, неравенства, прогрессии}$)
    * Геометрия ($\text{планиметрия, стереометрия}$)
    * Тригонометрия ($\sin$, $\cos$, $\tan$)
    * Функции и графики
    * Вероятность и статистика
    * Задачи с параметром (профильный уровень)
* **База задач** — более 500 заданий с решениями и подсказками.
* **Тренировочные тесты** — с таймером и автоматической проверкой ответов.
* **Разбор решений** — пошаговое объяснение к каждой задаче.
* **Личный кабинет** — отслеживание прогресса, статистика по темам, история попыток.
* **Адаптивный режим** — подбор задач по уровню сложности на основе предыдущих результатов.
